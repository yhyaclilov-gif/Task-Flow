# ⚡ TaskFlow

TaskFlow, tapşırıqlarınızı asanlıqla idarə etmək və izləmək üçün hazırlanmış minimalist bir veb tətbiqidir.

## 🌟 Əsas Xüsusiyyətlər
* **Google Authentication:** İstifadəçilər öz Google hesabları ilə sürətli şəkildə daxil ola bilərlər.
* **Tapşırıq İdarəetməsi:** Taskların yaradılması, redaktə edilməsi və silinməsi (CRUD).
* **Real-time Database:** Bütün məlumatlar verilənlər bazasında (Database) təhlükəsiz şəkildə saxlanılır.
* **Dark Tech Aesthetic:** Göz yormayan, müasir və qaranlıq interfeys dizaynı.

## 🛠 Texnologiyalar
Layihənin hazırlanmasında istifadə olunan əsas alətlər:
- **Frontend:** HTML5, CSS3, JavaScript
- **Auth:** Google Firebase / OAuth
- **Backend/Database:** Firebase Firestore 
- **Version Control:** Git & GitHub

## 🚀 İstifadə Qaydası
1. Proyekti brauzerdə açın.
2. "Google ilə daxil ol" düyməsinə klikləyərək autentifikasiyadan keçin.
3. Yeni tapşırıqlar əlavə edin və statuslarını dəyişdirin.
